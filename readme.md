# SMS压力测试工具

## ⚠️ 重要声明

**本项目仅供安全研究、教育学习和授权测试使用，严禁用于任何非法用途！**

## 项目简介

这是一个基于JavaScript的短信API测试工具，用于验证和测试短信服务接口的功能性和稳定性。

## 🚨 使用须知

### 合法用途
- ✅ 安全研究和技术学习
- ✅ 授权下的渗透测试
- ✅ 系统安全评估
- ✅ 教育目的演示

### 禁止行为
- ❌ 对未授权目标进行测试
- ❌ 骚扰他人或恶意轰炸
- ❌ 用于报复、敲诈等非法目的
- ❌ 商业滥用或垃圾信息发送

## ⚖️ 法律责任

使用者需对自身行为承担全部法律责任：
- 未经授权测试他人系统可能构成违法
- 滥用可能导致民事诉讼或刑事指控
- 请务必在合法合规的范围内使用

## 🔒 安全建议

1. **仅测试自有系统**：确保拥有测试目标的明确授权
2. **控制测试频率**：避免对服务造成实质性影响
3. **遵守服务条款**：尊重各平台的用户协议
4. **保护个人隐私**：不要泄露他人联系方式

## 🛠️ 技术特性

- 支持多种短信API接口
- 可配置请求频率和次数
- 模块化设计便于扩展
- 实时状态监控

## 📝 使用指南

1. 确保获得测试目标的明确授权
2. 配置合适的测试参数
3. 监控测试过程，避免过度请求
4. 测试完成后及时停止

在开始测试前，请仔细检查：
